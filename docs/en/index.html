What it is

A GUI for repeatable encoding workflows

FlowEncode is built for users who want a graphical Windows video encoding GUI without giving up the flexibility of command-line encoders, VapourSynth scripts and Av1an automation. It organizes input/output paths, encoder parameters, dependency state, queued jobs and logs into one desktop workflow.

Features

Core encoding modules

VapourSynth workspace

Edit .vpy and .py scripts, inspect diagnostics, preview scripts and review preview logs.

Video encoding queue

Configure x264, x265 and SVT-AV1 tasks with input/output paths, encoder parameters, command previews and real-time logs.

Av1an target quality

Run Av1an-based target-quality workflows with VMAF settings, probe counts, parallelism controls and extra encoder arguments.

Template library

Save, import, export, pin and reuse .profile templates for repeatable encoding presets.

First-run environment guide

Check, install, update or remove supported runtimes, encoders, plugins and workflow tools.

Local data by default

FlowEncode does not actively upload source files, output files, templates or local paths.

Workflow

From script to queued encode

  1. 1

    Prepare sources and scripts

    Use FFmpeg, FFprobe, VSPipe and VapourSynth assets in a managed workspace.

  2. 2

    Choose encoder settings

    Build x264, x265, SVT-AV1 or Av1an jobs with visible command previews.

  3. 3

    Run and inspect

    Execute queued jobs, watch logs, reuse templates and keep workflows reproducible.
